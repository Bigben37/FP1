\subsection{Bestimmung der Halbwertszeit von 40K}
Im letzten Versuchsteil wird $\beta$-Strahlung untersucht.
Dazu wird eine Aluminiumschale mit ca. 2\,g KCl befüllt und in das Zählrohr
eingesetzt. Die Zählrohrspannung wird in 100\,V-Schritten zwischen 2500\,V und
4000\,V variiert.
Anschließend wird eine Aktivitätsmessung bei 3200\,V Zählrohrspannung an der
selben Probe durchgeführt. Ein Schätzwert für die Aktivität bei dieser Spannung
ist aus obiger Messung schon vorhanden. Mit diesem Schätzwert lässt sich die
benötigte Messzeit berechnen, um einen relativen statistischen Fehler von
2\,\% auf die Aktivität zu erhalten. Die Messzeit beträgt 420\,s.
